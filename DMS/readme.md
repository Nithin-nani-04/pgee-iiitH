## 📌 2. Predicate Logic

### 🔹 Concepts

* Predicates
* Quantifiers: ∀ (for all), ∃ (exists)

### 🔹 Important

* Negation of quantifiers ⭐

  * ¬∀x P(x) = ∃x ¬P(x)
  * ¬∃x P(x) = ∀x ¬P(x)
